What's this all about?

Ptakopět (Překlad Tam A KOntrolně zPĚT) is a browser plugin for "outbound translation". It tackles the problem of translating to unknown languages when filling in online forms. Right now only English-Czech translation is supported, but that should change soon. This plugin was developed as a semestral project by Vilém Zouhar (with some help by Ondřej Bojar) and uses translation backed by Martin Popel (state of the art translation system). More info + addon.
